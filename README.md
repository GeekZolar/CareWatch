## ğŸš€ Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd ltc-website
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

```bash
npm run preview
```

## ğŸ“ Content Areas

### Medical Tests
- Imaging Tests (X-Ray, CT Scan, MRI, Ultrasound)
- Diagnostic Tests (EKG/ECG, Stress Test)
- Blood Tests (CBC, Chemistry Panel, Cholesterol, Glucose)
- Urine Tests (Urinalysis, Culture)

### Equipment
- Monitoring Equipment (BP monitors, EKG, Thermometer, Pulse Oximeter)
- Respiratory Equipment (Oxygen Concentrator, Nebulizer, CPAP/BiPAP)
- Infusion & IV Equipment
- Beds & Positioning Equipment

### Medical Specialists
- Rehabilitation Specialists (Physiotherapist, OT, Speech Therapist)
- Medical Physicians (Geriatrician, Cardiologist, Neurologist)
- Nursing & Support Care
- Other Specialists (Ophthalmologist, Audiologist, Dentist)

- Transfer & Safety Equipment
- Lifting & Positioning Equipment
- Daily Living Aids

- Special Diet Types (Soft, Diabetic, Low-Sodium, Renal, High-Protein, Pureed)
